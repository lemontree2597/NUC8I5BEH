# NUC8I5BEH Hackintosh

This is the hackintosh for the NUC8I5BEH

- OS:macos 12.0 Beta(21A5268h)
- OpenCore 0.7.0
- CPU: Intel Core i5-8259U
- SSD: 500GB Crucial SSD
- RAM: 16GB  Team 2400
- WIFI&Bluetooth: BCM94360CS2
- Monitor: AOC

# BIOS

- Disable **Secure Boot**
- Enable **Legacy Boot**
- Disable **Wi-Fi and Bluetooth**
